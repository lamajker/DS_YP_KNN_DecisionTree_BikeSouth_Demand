# DS_YP_KNN_DecisionTree_BikeSouth_Demand

# Нелинейные модели против южной погоды

Проект по прогнозированию почасового спроса на аренду велосипедов для компании BikeSouth.

## Цель проекта

Построить модель машинного обучения, которая прогнозирует количество арендованных велосипедов за час точнее, чем baseline-модель на основе линейной регрессии.

## Задачи

- изучить качество baseline-модели;
- провести исследовательский анализ данных;
- обработать пропуски в погодных признаках;
- подготовить признаки для моделей;
- обучить KNeighborsRegressor и DecisionTreeRegressor;
- подобрать гиперпараметры с помощью Optuna;
- сравнить модели по RMSE, MAE и R2;
- выбрать лучшую модель и проверить её на тестовой выборке;
- сохранить финальный пайплайн.

## Данные

В проекте используются файлы:

- `baseline_linear_regression_pipeline.pkl` - готовая baseline-модель;
- `ds_s14_train_data.csv` - обучающая выборка;
- `ds_s14_test_data.csv` - тестовая выборка.

Целевая переменная:

- `Rented Bike Count` - количество арендованных велосипедов за час.

Основные признаки:

- температура;
- влажность;
- скорость ветра;
- видимость;
- точка росы;
- солнечная радиация;
- дождь и снег;
- сезон;
- праздничный день;
- рабочий день сервиса;
- признаки времени суток.

## Лучшая модель

Лучшей моделью стала оптимизированная `KNeighborsRegressor`.

Гиперпараметры:

```python
n_neighbors = 11
weights = "distance"
p = 1
````

Модель показала качество на тестовой выборке лучше, чем на кросс-валидации, что говорит об отсутствии признаков сильного переобучения.

## Основные выводы

Спрос на аренду велосипедов сильно зависит от погодных условий, сезона и времени суток.

Наиболее важные факторы:

* температура;
* влажность;
* сезон;
* время суток;
* наличие дождя или снега;
* солнечная радиация;
* рабочий день сервиса.

Оптимизированная kNN-модель хорошо подходит для этой задачи, потому что ищет в истории похожие погодные и временные ситуации и делает прогноз на основе ближайших наблюдений.

Категориальные признаки осадков `rain_level` и `snow_level` оказались полезнее, чем исходные числовые значения дождя и снега. Для бизнеса важнее сам факт и сила осадков, а не точное количество миллиметров.

## Практическая ценность

Модель может помочь BikeSouth:

* точнее планировать спрос;
* заранее готовиться к пиковым часам;
* учитывать снижение спроса при плохой погоде;
* эффективнее распределять велосипеды между станциями;
* снижать простой и дефицит велосипедов.
